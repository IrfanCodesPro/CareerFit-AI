## 🌟 Overview

CareerFit AI is an AI-powered Android application developed by **Nexora Helix** that helps students and professionals analyze resumes, improve ATS compatibility, identify skill gaps, receive personalized career recommendations, and prepare for technical interviews using Google's Gemini AI.
