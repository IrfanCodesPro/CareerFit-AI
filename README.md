📖 Overview
CareerFit AI is an AI-powered Android application developed by Nexora Helix to help students and professionals make smarter career decisions. Using Google Gemini AI, the application analyzes resumes, evaluates ATS compatibility, identifies skill gaps, recommends career paths, and provides personalized interview preparation.
🎯 Objectives
Build AI-powered career guidance.
Improve ATS compatibility.
Recommend suitable job roles.
Detect skill gaps.
Generate learning roadmaps.
Prepare users for technical and HR interviews.
