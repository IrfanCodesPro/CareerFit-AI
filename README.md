# Overview

CareerFit AI is an AI-powered Android application developed by **Nexora Helix** to help students and professionals navigate their career journey with confidence. The platform combines intelligent resume analysis, Applicant Tracking System (ATS) optimization, AI-driven career recommendations, personalized learning roadmaps, and technical interview preparation into a single, seamless experience.

Leveraging **Google Gemini AI**, **Kotlin**, **Node.js**, and **MongoDB Atlas**, CareerFit AI delivers actionable insights by analyzing resumes, identifying strengths and skill gaps, recommending suitable career paths, and guiding users toward industry-ready opportunities.

# Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/CareerFit-AI.git
```

## Navigate to Project

```bash
cd CareerFit-AI
```

## Configure

- Configure Google Gemini API Key
- Configure MongoDB Atlas Database
- Configure Backend Base URL

## Build & Run

Open the project using **Android Studio**, sync Gradle dependencies, and run the application on an Android device or emulator.
